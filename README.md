# HABC

Official implementation of:

**HABC: Hierarchical Advantage Weighting for Online RL Fine-Tuning of VLAs from Sparse Episode Outcomes**

This repository will host the offline SFT code, online RL post-training code, evaluation scripts, and deployment code for HABC.

The implementation is based on [OpenPI](https://github.com/Physical-Intelligence/openpi).

Project page: https://acerobotics-vla.github.io/HABC-Website/

Code will be released soon.
